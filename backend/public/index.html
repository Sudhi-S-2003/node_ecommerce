Products API

Store and request product data for your frontend experience.

Use the following endpoints to interact with the API:

  • GET /api/products - Retrieve a list of all products.
  • GET /api/product/:id - Retrieve a single product by ID.
  • POST /api/product - Create a new product.
  • PUT /api/product/:id - Update a single product by ID.
  • DELETE /api/product/:id - Delete a single product by ID.

Retrieve a list of all products.


function displayProducts() {
fetch('http://localhost:8000/api/products')
.then(response => response.json())
.then(data => displayData(data))
.catch(error => console.error('Error fetching products:', error));
}

Retrieve a single product by ID.


function displayProduct(id) {
fetch(`http://localhost:8000/api/product/${id}`)
.then(response => response.json())
.then(data => displayData(data))
.catch(error => console.error(`Error fetching product with id ${id}:`, error));
}

Create a new product.


function createProduct(product) {
fetch('http://localhost:8000/api/product', {
method: 'POST',
headers: {
'Content-Type': 'application/json'
},
body: JSON.stringify(product)
})
.then(response => response.json())
.then(data => console.log('Product created:', data))
.catch(error => console.error('Error creating product:', error));
}

Update an existing product.


function updateProduct(id, product) {
fetch(`http://localhost:8000/api/product/${id}`, {
method: 'PUT',
headers: {
'Content-Type': 'application/json'
},
body: JSON.stringify(product)
})
.then(response => response.json())
.then(data => console.log('Product updated:', data))
.catch(error => console.error('Error updating product:', error));
}

Delete a product.


function deleteProduct(id) {
fetch(`http://localhost:8000/api/product/${id}`, {
method: 'DELETE'
})
.then(response => response.json())
.then(data => console.log('Product deleted:', data))
.catch(error => console.error('Error deleting product:', error));
}
